# CodeCraft | Crafting Digital Solutions

CodeCraft is a responsive, modern landing page template designed to showcase digital solutions, web development services, and projects. Built with **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**, it features a mobile-first, fully responsive design.

## Features
- Fully responsive design for mobile, tablet, and desktop
- Hero section with engaging headline and call-to-action buttons
- Mobile-friendly navigation menu with toggle animation
- About, Services, Projects, and CTA sections
- Footer with company links, resources, legal information, and social media links
- Smooth hover and transition effects for buttons and cards
- Easy-to-customize Tailwind CSS utility classes

---

## Technologies
- **HTML5** – Semantic markup for structure
- **Tailwind CSS** – Utility-first styling framework
- **JavaScript** – For mobile menu toggle functionality
- **Flexbox & CSS Grid** – Layout management
- **Responsive Web Design** – Mobile-first approach

```bash

```
2. Navigate to the project directory:
```bash
cd codecraft
```
3. Navigate to the project directory:

Open index.html in your preferred web browser.

## Usage

Open index.html to view the fully responsive CodeCraft landing page.

Modify the HTML and CSS as needed to customize branding, images, and content.

Replace placeholder images in the images folder with your own assets.
